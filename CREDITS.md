# Credits

Every third-party asset in this repo gets a line here in the same commit that adds it.
Treat this file as an SBOM for assets. An asset with no entry here is a bug.

Format: `filename | source URL | author | licence`

## Art

All art below is from Kenney (kenney.nl), released under CC0 1.0 (public domain).
Attribution is not required by the licence, but is recorded here as an asset SBOM.
Each file is a single tile or sprite extracted from a Kenney pack and renamed; the
original pack tile index or filename is noted for provenance.
